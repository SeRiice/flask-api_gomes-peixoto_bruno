Booking Hôtel

Bienvenue, ceci est une API permettant de gérer la réservation de chambres d'hôtel pour une chaine d'hôtels.

Voici les différentes routes disponibles :

  • /api/chambres/disponibles
    • GET
  • /api/reservations
    • POST
  • /api/reservations/{id}
    • DELETE
  • /api/chambres
    • POST
  • /api/chambres/{id}
    • PUT, DELETE
